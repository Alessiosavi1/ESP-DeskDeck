# ESP-DeskDeck

> A compact DIY desk companion and macro-pad powered by **ESP32-C3 Mini**.

ESP-DeskDeck combines:

- 📶 **Wi-Fi** for environmental data retrieval
- 🔵 **Bluetooth LE** for wireless desktop interaction
- 🧩 A compact form factor for daily desk workflows

---

## ✨ Overview

ESP-DeskDeck is designed as a practical bridge between the physical and digital world:  
fast hardware controls + useful live data in a clean desktop setup.

## 🚀 Features

- Compact and customizable desk device
- Wireless communication via BLE
- Wi-Fi connectivity for external data
- Ideal for productivity and automation use-cases

## 🛠️ Hardware

- ESP32-C3 Mini
